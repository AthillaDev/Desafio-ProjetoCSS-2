<b>Projeto - Aprimoramento de halibidades</b>


Este repositório contém um projeto simples criado para demonstrar conceitos básicos de HTML5 e CSS3 , ideal para iniciantes que desejam entender a estrutura de uma página web e como aplicar estilos fundamentais.

🔗 <b>Demonstração online:</b>

https://github.com/AthillaDev/Desafio-ProjetoCSS-2

📚 <b>Sobre o projeto</b>
Este projeto foi desenvolvido para fins de estudo do HTML5 e CSS, apresentando:

Estrutura básica de um documento HTML5
Uso de tags semânticas ( header, section, footer, etc.)
Inclusão de imagens
Links e navegação simples
Estilização com CSS externo
Layout responsivo básico
É um ótimo ponto de partida para quem está começando no mundo do desenvolvimento web.

🗂 <b>Estrutura do projeto</b>

Projeto-Introdução-HTML-e-CSS/
├─ index.html
├─ style.css

🧱 <b>Tecnologias utilizadas</b>
HTML5 → estrutura e conteúdo
CSS3 → estilização, núcleos, tipografia e layout
Nenhuma estrutura foi utilizada. Todo o código é nativo (puro).



📌 <b>Possíveis melhorias</b>

Criar mais balões usando tags semânticas

Aplicar conceitos de responsabilidade mais avançados
